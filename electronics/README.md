## Configuração GPIOs

### Ponte H

| Ponte H L9110 | ESP32    |
| ------------- | -------- |
| B-1A          | D5       |
| B-1B          | D23      |
| G             | GND      |
| VCC           | Vin (5v) |
| A-1A          | D13      |
| A-1B          | D12      |
